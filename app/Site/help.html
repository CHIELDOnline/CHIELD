Correcting data on CHIELD

Have you spotted a problem with some of the data in CHIELD? You can fix it! You'll need a free github account.

Go to the document page where you see the problem. You can:

  • Click "Edit data" to directly edit the data through the CHIELD website.
  • Click "Raise Issue" to write a note (log into github first)
    • Click on "Source" to see the source data on github. You can fork and edit the repository directly.

Further help

If you spot a technical problem, you can raise an issue on the GitHub page. Or you can email the developers (see the about page).
